Обо мне и мнение о GitHub, Figma, HTML/CSS - JavaScript, StackBlitz, CodePen

Мне 14 лет, увлекаюсь графическим программированием, WebGL и созданием 3D графики. Этот сайт - моя дипломная работа. В этом разделе я бы хотел бы рассказать о платформах, и не только. GitHub - Мой главный рабочий инструмент. Хранить код на GitHub - это как вести цифровой дневник программиста. Видно весь прогресс: от первых простых скриптов до сложных проектов. Особенно нравится, что можно смотреть, как другие решают задачи - это лучший способ учиться. Для диплома GitHub идеален - вся история изменений видна, можно показать, как проект развивался. Figma - Для визуалов и дизайнеров это просто находка. Когда делал макет для диплома, понял главное преимущество: можно быстро перебирать варианты без переписывания кода. Интерфейс интуитивный, даже в 14 лет разобрался быстро. Единственное - нужно понимать, что красивый макет в Figma ≠ работающий сайт. Это только первый шаг. HTML/CSS + JavaScript - Основа всего, что видишь в браузере. HTML - это каркас дома, CSS - отделка и дизайн, а JavaScript - электричество и водопровод. Начинал с простых страничек, теперь делаю интерактивные проекты. Главный плюс - результат видно сразу. Минус - в JavaScript много "подводных камней", особенно в асинхронных операциях. StackBlitz - Обнаружил недавно, когда нужно было показать 3D-проект. Главное преимущество: не нужно ничего устанавливать. Полноценная среда разработки прямо в браузере - это волшебно. Особенно удобно для быстрых демонстраций или когда нужно поработать с чужим кодом. Для новичков - идеальный вариант, чтобы не мучиться с настройкой окружения. CodePen - Мой цифровой скетчбук. Когда приходит идея или нужно потестировать небольшой кусок кода, открываю CodePen. Интерфейс минималистичный, ничего лишнего. Часто смотрю, что делают другие - можно найти интересные решения и техники. Минус - для больших проектов не очень подходит, больше для экспериментов. Личное мнение: В 14 лет эти инструменты дают невероятные возможности. 10 лет назад нужно было бы устанавливать кучу программ, сейчас всё в браузере. GitHub учит дисциплине (коммиты, ветки), Figma - дизайн-мышлению, HTML/CSS/JS - основам программирования, а StackBlitz и CodePen позволяют экспериментировать без риска что-то сломать. Идеальный стек для старта в веб-разработке. Для дипломного проекта эта комбинация идеальна: Figma для макета, GitHub для хранения кода, StackBlitz для демо-версий, а HTML/CSS/JS как основной инструмент реализации.
