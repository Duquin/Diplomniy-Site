Чему я научился

За процесс обучения и создание проектов я освоил ключевые технологии и методологии веб-разработки:

JavaScript

Разработка интерактивных интерфейсов, работа с DOM, асинхронные запросы, ES6+ синтаксис

HTML5 & CSS3

Семантическая верстка, Flexbox, Grid, адаптивный дизайн, CSS-анимации, препроцессоры

React.js

Разработка SPA-приложений, работа с состояниями, компонентный подход, React Hooks

Git & GitHub

Контроль версий, ветвление, пулл-реквесты, деплой проектов на GitHub Pages

UI/UX Design

Работа с Figma, создание прототипов, проектирование пользовательских интерфейсов

Оптимизация

Улучшение производительности, lazy loading, минификация, оптимизация изображений

Создание веб-проектов научило меня не только писать код, но и мыслить как пользователь, предвидеть потребности и создавать интуитивно понятные интерфейсы. Каждый проект — это новый вызов и возможность применить современные подходы к разработке.
